# Change Log
All notable changes to the "git patch" extension will be documented in this file.

## [1.0.0] - 2026-01-09
### Added
- **Clipboard Support**: New commands to copy patches to clipboard and apply patches from clipboard
- Modern VSCode extension APIs with updated dependencies
- Improved development setup with latest TypeScript and testing frameworks

### Changed
- Major dependency updates for better performance and compatibility
- Updated to VSCode engine ^1.96.0 for modern API support

## [0.2.0] - 2018-02-05
### Added
- Apply patch either from '.patch' Or '.diff'.
- Better integration with dialog that allows you to Save Patch at different location & Apply from custom location.
